# AnyFile

The next generation of file manipulation. Cinematic, client-side, flow-state efficient.

## Installation
1. `npm install`
2. `npm run dev`

## Features
- **Flow UI**: Liquid glass aesthetics with parallax and tilt.
- **Material Design**: High-res abstract material wallpapers that auto-rotate.
- **Sonic Interaction**: Subtle, synthesized sound effects for tactile feedback.
- **Client-Side Core**: Heavy processing (Merge, Split, Watermark) done in-browser via WebAssembly/JS.
- **Dark/Light Mode**: Sophisticated theming system.

## Tech Stack
- React 19
- TailwindCSS
- Framer Motion
- PDF-Lib
- Lucide React
